How many page types?

The whole site runs on two page types. The Home page does almost everything by changing state — it is not many separate pages.

1 · The Home page built

One adaptive page (index.html). It has four states: Index → Category → Collection → Detail. Wireframes A–D below.

2 · Commission page built

A standalone page for a single commission. One exists today: Viceroy — The Observatory (observatory.html). Wireframe E.

The flow — one grammar, repeated

Every box is either a collection (fans out into more boxes) or a leaf (expands into a detail view). The same gesture repeats at every depth.

Every step is reversible — clicking back fans the boxes in again. Step C only appears for boxes that contain more boxes (e.g. Surface Surveys).
